Executive Summary

 

Arkansas is unique in that it can provide film producers a variety of different types of natural settings without requiring much travel. Arkansas has also been the filming location for some famous movies. But unfortunately, not enough film producers seem aware of the benefits of filming in Arkansas, or of the connections between Arkansas and famous works. This project aim is to rectify these issues by creating a website that will feature Arkansas’s connection to the filming industry.

 

Project Purpose/Justification

 

Business Need/Case

 

Arkansas is very diverse in the type of natural settings it can provide film producers, making it unique. This diversity of natural settings would ultimately save film producers money by reducing the amount of travel necessary for them to complete their projects. In addition, Arkansas already has some recognition within the filming industry, seeing as how famous movies such as the cult classic sling blade were filmed in Arkansas.

 

Business Objectives

 

The business objectives for this project are in direct support of our strategic plan to obtain a higher level of interest, in Arkansas, by film producers, in order to ultimately get a measurable increase in the number of films being produced in the state of Arkansas.

 

-        Design a new website promoting Arkansas as a prime filming location by the end of capstone 1.

-        Build, test, and deploy the website by the end of capstone 2.

-        Increase the amount of films produced in Arkansas by x amount over y years.

 

Project Description

 

The website we intend to build will showcase Arkansas’s potential as a filming location by allowing users to search for and obtain information about Arkansas and the films produced within it, in an attempt to increase the amount of films made in Arkansas. The website will achieve this by querying data from an established database, via an API, and then neatly displaying the retrieved data onto the website.

Requirements

 

This project must meet the following list of requirements in order to achieve success. 

-        Users must be able to navigate the website using a mobile device.

-        The Website must be responsive and not take too long to load pages.

-        The website must be able to interact with the DB.

-        The website must be able to display data from the DB.

-        The website must be able to format the data from the DB.

-        The data on the website must be clearly displayed.

Additional requirements may be added as necessary, with project sponsor approval, as the project moves forward.

 

Constraints

Assumptions

 

The following are a list of assumptions.  Upon agreement and signature of this document, all parties acknowledge that these assumptions are true and correct:

-        The purpose of this project will be communicated through the client prior to deployment.

-        The database will be kept up to date.

-        The data in the database is accurate.

 

Preliminary Scope Statement

 

This project involves designing, building, and testing a website whose primary function is to retrieve data from a preexisting database, by means of an API, and then neatly display it. The website will be built using programming Languages such as HTML5, CSS, PHP, and Javascript. The designing of the Arkansas Movie Database website will take place during capstone 1. While the construction and testing of the website will take place during capstone 2. The Arkansas Movie Database should be fully deployed by the end of capstone 2. A presentation and report are expected at the end of each semester to evaluate the projects progress and success.

 

 

 

Risks

 

The following risks for the Arkansas Movie Database project have been identified.  The project manager will determine and employ the necessary risk mitigation/avoidance strategies as appropriate to minimize the likelihood of these risks: 

-        If we lose access to the database, then our website is pretty much useless.

-        If the data in the database is not correct there could be liability issues.
